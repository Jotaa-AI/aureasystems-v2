# Banco de temas para el blog

> Lista curada de ideas long-tail para el agente generador. Editable por humanos.
>
> **Cómo añadir un tema**: copia el bloque, asigna `priority` (1-5, 5 = urgente),
> `category` (de la lista válida) y deja `status: pending`.
>
> **Cómo el agente lo consume**: cuando usa un tema, cambia `status: used`,
> añade `used_date` y `used_in: {slug-del-post}`. NO lo borra (sirve de histórico).
